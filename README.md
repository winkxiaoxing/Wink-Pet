# WinkPet

WinkPet，适配多系统的桌面宠物。透明小窗常驻桌面，支持闲逛走动、点击互动、拖拽挪位与自定义形象，并可从托盘快捷管理。

创作者：[wink小星](https://github.com/winkxiaoxing)

喜欢的话欢迎点右上角 ⭐ Star，谢谢支持。

---

## 下载

请从 **[Releases](https://github.com/winkxiaoxing/Wink_Pet/releases)** 获取最新安装包：

| 文件 | 说明 |
|------|------|
| `WinkPet Setup x.x.x.exe` | **推荐**：安装版，支持开机启动 |
| `WinkPet x.x.x.exe` | 便携版，解压即可用 |

> 若 Windows 提示「已保护你的电脑」：点击 **更多信息** → **仍要运行**（当前安装包未做代码签名，属常见提示）。

## 功能一览

- 透明悬浮桌宠，不挡日常操作  
- 自主闲逛 / 站立，单击、双击、三连点互动  
- 拖拽移动，可选边缘吸附  
- 自定义形象与多宠并行  
- 托盘常驻：打开设置、退出  

---

## 怎么用

- **右键宠物**：打开设置、切换形象、锁定位置、退出  
- **系统托盘**（任务栏右下角）：打开设置 / 退出  
- **单击 · 双击 · 三连点**：互动 / 开心 / 委屈  
- **开机启动**：用 Setup 安装后，在设置 → **启动** 中打开  

---

## 常见问题

**看得见但点不着？**  
把鼠标移到宠物身上再点；或在设置 → 启动里关闭「点击穿透」。也可点托盘打开设置。

**自定义形象？**  
设置 → **形象**：新建或导入形象包。

**配置文件在哪？**  
`%APPDATA%\wink-desktop-pet\`
